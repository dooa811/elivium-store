# ELIVIUM — Luxury Fashion Store

> A full-stack e-commerce platform for luxury fashion.

🌐 **Live Demo:** [elivium-store.vercel.app](https://elivium-store.vercel.app)

---

## ✨ Features

- Email verification & JWT authentication
- Product catalog with search & filters
- Shopping cart & wishlist
- Order management
- Admin dashboard (products, orders, statistics)
- Fully responsive design

---

## 🛠️ Tech Stack

| Frontend | Backend | Database |
|----------|---------|----------|
| React + Vite | Node.js + Express | Supabase (PostgreSQL) |
| Tailwind CSS | JWT Auth | Railway (hosting) |
| Framer Motion | SendGrid (email) | Vercel (hosting) |
